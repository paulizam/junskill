## 风格1：chinese_classic 中国风

### A. 核心意象词库
- **器物类**：素胚、青花、釉色、宣纸、毛笔、墨色、瓷器、琉璃、铜镜、砚台、折扇、琵琶、古筝
- **自然类**：烟雨、芭蕉、骤雨、月色、炊烟、檀香、落花、秋水、浮云、古道、晚风、霜、露、红豆
- **人文类**：仕女图、泼墨山水、行书、楷书、茶经、本草、碑文、汉隶、兰亭、画卷
- **颜色类**：天青色、朱红、丹青、素白、墨黑、翠绿、鹅黄
- **地点类**：江南小镇、长安、洛阳、古道、长城、敦煌、西湖、秦淮河
- **文化类**：法典、楔形文字、经卷、咒语、年轮、朝代、轮回、红尘、三生、宿命

### B. 经典句式模板
1. `[材质/颜色] + [动作] + 出 + [意象]，[人物动作] + [情感]`
   → "素胚勾勒出青花笔锋浓转淡 瓶身描绘的牡丹一如你初妆"
2. `[自然现象] + 等 + [天气]，而我 + 在等 + [你]`
   → "天青色等烟雨 而我在等你"
3. `[器物/地点] + 惹 + [自然现象]，而我 + 惹了 + [你]`
   → "帘外芭蕉惹骤雨门环惹铜绿 而我路过那江南小镇惹了你"
4. `[人物] + 的 + [意象] + 已 + [变化动词]`
   → "你的笑容已泛黄"
5. `[宏大意象] + 如 + [比喻]，我 + 只 + [渺小选择]`
   → "繁华如三千东流水 我只取一瓢爱了解"
6. `[时间] + 的 + [风沙/痕迹]，[现在进行时]`
   → "唐朝千年的风沙 现在还在刮"
7. `[感官细节]，[动作] + 偷/拆/惹 + [抽象概念]`
   → "水向东流 时间怎么偷"

### C. 韵脚偏好
- 主韵：-an/-en（淡/然/半/人/山/远/烟/天/见）
- 辅韵：-i（衣/雨/里/你/细/泥）
- 换韵模式：主歌A韵 → 副歌B韵

### D. 节奏与结构特征
- 句式偏长（7-12字为主），大量使用对仗
- "素颜韵脚诗"特征：不用标点，断行即断句
- 副歌核心句型重复2次，末句收尾

### E. 参考金句
- "天青色等烟雨 而我在等你 炊烟袅袅升起 隔江千万里" ——《青花瓷》
- "谁在用琵琶弹奏 一曲东风破 岁月在墙上剥落 看见小时候" ——《东风破》
- "繁华如三千东流水 我只取一瓢爱了解 只恋你化身的蝶" ——《发如雪》
- "雨纷纷 旧故里草木深 我听闻 你始终一个人" ——《烟花易冷》
- "狼牙月 伊人憔悴 我举杯 饮尽了风雪" ——《发如雪》

## 风格2：dark_gothic 暗黑哥特

### A. 核心意象词库
- **生物类**：吸血鬼、女巫、蝙蝠、乌鸦、蜘蛛、蟑螂、蚂蚁、蜻蜓、秃鹰、狼人、半兽人
- **场景类**：古堡、教堂、墓地、废墟、壁炉、走廊、地窖、钟楼、森林、荒野
- **物件类**：蜡烛、银制茶壶、水晶球、扫把、十字架、棺材、玫瑰、银十字
- **氛围类**：雾气、夜色、冰冷、灰暗、诡异的安静、腐烂、腐肉、嗜血、诅咒
- **宗教类**：福音、祭司、祈祷、诅咒、占卜、法典、灵魂、忏悔、弥赛亚

### B. 经典句式模板
1. `[暗黑生物] + [诡异行为]，[反常结果]`
   → "藤蔓植物爬满了伯爵的坟墓"
2. `不会 + [正常行为] + 的 + [角色]，用 + [反常方式] + [动作]`
   → "不会骑扫把的胖女巫 用拉丁文念咒语"
3. `[时间/光线] + 射进/照进 + [地点]，能 + [超自然结果]`
   → "下午三点阳光射进教堂的角度 能知道你前世是狼人还是蝙蝠"
4. `失去你 + [负面感受]，[暗黑意象] + [蔓延]`
   → "失去你 泪水混浊不清 失去你 我连笑容都有阴影"
5. `[自然现象] + [拟人化暗黑描写]，嘲笑/讽刺 + [情感]`
   → "风在长满青苔的屋顶 嘲笑我的伤心"

### C. 韵脚偏好
- 主韵：-u（墓/呜/呼/猪/步/户/烛/屋）
- 辅韵：-ing/-eng（冷/等/醒/声/影/明）
- 特色：AABB式押韵，制造恐怖童谣效果

### D. 节奏与结构特征
- Rap段落频繁出现
- 短句节奏（3-5字）+ 长句叙事交替
- 常用叠词/拟声词（"恍恍惚惚"、"啦啦啦呜"、"哼哼哈兮"）

### E. 参考金句
- "不会骑扫把的胖女巫 用拉丁文念咒语啦啦呜" ——《威廉古堡》
- "为你弹奏萧邦的夜曲 纪念我死去的爱情" ——《夜曲》
- "一群嗜血的蚂蚁 被腐肉所吸引 我面无表情 看孤独的风景" ——《夜曲》
- "再也没有纯白的灵魂 自人类堕落为半兽人" ——《半兽人》
- "当鸽子不再象征和平 我终于被提醒 广场上喂食的是秃鹰" ——《夜曲》

## 风格3：romance_sweet 甜蜜情歌

### A. 核心意象词库
- **食物类**：糖果、咖啡、草莓、番茄、果酱、鸡蛋糕、麦芽糖、甜筒、蜂蜜、七里香
- **自然类**：阳光、蒲公英、彩虹、花瓣、蝴蝶、薰衣草、珊瑚、海风、蜻蜓、麻雀
- **日常类**：气球、手机、日记、信封、入场券、折纸、秋千、单车、铅笔、纸飞机
- **氛围类**：温暖、微笑、甜蜜、害羞、心动、脸红、初恋、撒娇、幸福

### B. 经典句式模板
1. `我轻轻地 + [亲密动作]，你 + [回应/状态]`
   → "我轻轻地尝一口你说的爱我"
2. `[地点/场景] + [甜蜜细节]，我 + [浪漫行为] + [对方]`
   → "塞纳河畔 左岸的咖啡 我手一杯 品尝你的美"
3. `[自然景物] + 像 + [甜蜜比喻]，[可爱反应]`
   → "那温暖的阳光 像刚摘的鲜艳草莓"
4. `我 + [笨拙动作] + 只为 + [对方]，[甜蜜结果]`
   → "我顶着大太阳 只想为你撑伞"
5. `你说 + [可爱话语]，我 + [心动反应]`
   → "你说你有点难追 想让我知难而退"

### C. 韵脚偏好
- 主韵：-ei（谁/味/美/回/水/飞/泪）
- 辅韵：-an（甜/天/边/间/脸/远）

### D. 节奏与结构特征
- 句式中等偏短（5-8字为主）
- 副歌重复核心甜蜜句2-3次
- 结尾常以"愿意"/"在一起"收束

### E. 参考金句
- "塞纳河畔 左岸的咖啡 我手一杯 品尝你的美" ——《告白气球》
- "雨下整夜 我的爱溢出就像雨水" ——《七里香》
- "小学篱笆旁的蒲公英 是记忆里有味道的风景" ——《蒲公英的约定》
- "琥珀色黄昏像糖在很美的远方" ——《园游会》
- "薄荷色草地芬芳像风没有形状" ——《园游会》

## 风格4：romance_sad 伤感情歌

### A. 核心意象词库
- **天气类**：灰色天空、雨、风、寒风、雾、阴霾、雷、暴风、雪
- **痕迹类**：泛黄的日记、抽屉、旧照片、眼泪、回忆、伤口、疤痕
- **距离类**：隔着、远方、转身、离开、背影、后视镜、距离、海岸线
- **情感类**：遗憾、后悔、心碎、无法释怀、来不及、错过
- **对比类**：曾经/如今、靠近/远离、拥有/失去、记住/遗忘

### B. 经典句式模板
1. `[环境描写] + 像 + [悲伤比喻]`
   → "天空灰得像哭过"
2. `我知道 + [无可奈何的事实]，只是 + [固执/不舍]`
   → "我知道你我都没有错 只是忘了怎么退后"
3. `[感官细节] + [记忆触发]，[时间流逝] + 被 + [动作词]`
   → "抽屉泛黄的日记 榨干了回忆"
4. `你 + [离开行为]，我 + [残留感受]`
   → "你转身向北 侧脸还是很美"
5. `海鸟跟鱼相爱 / + [不可能的组合]，只是 + [意外/遗憾]`
   → "海鸟跟鱼相爱 只是一场意外"
6. `最美的 + [美好事物]，[否定/遗憾]`
   → "最美的爱情 回忆里待续"

### C. 韵脚偏好
- 主韵：-ong（空/风/中/动/痛/控/梦/冻）
- 辅韵：-ai（白/奈/开/海/哀）
- 换韵：主歌-ai → 副歌-ong

### D. 节奏与结构特征
- 长短句交替（情感充沛处短句爆发，回忆处长句铺陈）
- 副歌多用对比结构（过去的美好 vs 现在的遗憾）
- 结尾常为未完成的期待或认命

### E. 参考金句
- "天空灰得像哭过 离开你以后 并没有更自由" ——《退后》
- "海鸟跟鱼相爱 只是一场意外" ——《珊瑚海》
- "后视镜里的世界 越来越远的道别" ——《一路向北》
- "给你的爱一直很安静 来交换你偶尔给的关心" ——《一直很安静》
- "我受够了等待 你所谓的安排" ——《倒带》

## 风格5：wuxia_kungfu 武侠功夫

### A. 核心意象词库
- **武术类**：双截棍、龙拳、太极、铁砂掌、金钟罩、轻功、飞檐走壁、丹田、马步
- **武器类**：刀、剑、枪、弓、暗器、匕首、双刀、掌、拳
- **精神类**：仁者无敌、一身正气、傲骨、侠义、忠义、不屈、豪气、义气
- **场景类**：嵩山、少林、武当、江湖、战场、沙场、边关、客栈、古道、驿站
- **历史类**：长城、敦煌、黄河、泰山、民族、血脉、华夏、中原
- **中药类**：马钱子、决明子、苍耳子、莲子、山药、当归、枸杞

### B. 经典句式模板
1. `我 + [武术动作] + [宏大意象]，[豪迈结果]`
   → "我右拳打开了天 化身为龙"
2. `[地名] + 为圆心/起点，[民族/历史意象] + [壮阔描写]`
   → "以敦煌为圆心的东北东 这民族的海岸线像一支弓"
3. `快使用 + [武器/招式]，[拟声词/气势词]`
   → "快使用双截棍 哼哼哈兮"
4. `[武术口诀/气息描写]，[身心状态]`
   → "呼吸吐纳心自在 气沉丹田手心开"
5. `剑出鞘 / 刀出鞘，[恩怨了断]，[江湖态度]`
   → "剑出鞘 恩怨了 谁笑 我只求今朝 拥你入怀抱"

### C. 韵脚偏好
- 主韵：-ang（长/刚/当/枪/上/强/闯/狂/场）
- 辅韵：-ong（风/空/动/同/中/龙）
- 特色：口诀/拟声词段落不押韵，以节奏感为主

### D. 节奏与结构特征
- Rap段落极多，节奏密集（2-4字短词连击）
- 大量使用武术口诀和拟声词
- 副歌口号式重复（如"哼哼哈兮"）

### E. 参考金句
- "我右拳打开了天 化身为龙 把山河重新移动 填平裂缝" ——《龙拳》
- "快使用双截棍 哼哼哈兮 习武之人切记 仁者无敌" ——《双截棍》
- "天涯的尽头是风沙 红尘的故事叫牵挂" ——《红尘客栈》
- "剑出鞘 恩怨了 谁笑 我只求今朝 拥你入怀抱" ——《红尘客栈》

## 风格6：campus_youth 校园青春

### A. 核心意象词库
- **校园类**：课桌椅、黑板、操场、教室、走廊、图书馆、校服、考卷、铃响、三年二班
- **活动类**：折纸飞机、荡秋千、捞鱼、园游会、乒乓球、篮球、毕业典礼、罚站
- **人物类**：同桌、老师、小姑娘、同学、外婆、爷爷、朋友
- **自然类**：蜻蜓、蝉、蒲公英、夕阳、落叶、春风、阳光、白云
- **情感类**：纯真、暗恋、约定、成长、回忆、青春、梦想、第一次

### B. 经典句式模板
1. `[校园场景] + 的 + [小物件]，是 + [有味道/感觉的] + [回忆]`
   → "小学篱笆旁的蒲公英 是记忆里有味道的风景"
2. `[青春活动] + 我们 + [纯真行为]，[多年后的感慨]`
   → "将愿望折纸飞机寄成信"
3. `为什么 + [青春困惑]，为什么 + [反差对比]`
   → "为什么上课时举手很难 为什么拿在线宝物简单"
4. `一起 + [约定] + 的约定，[如今的感慨]`
   → "一起长大的约定 那样清晰 打过勾的我相信"
5. `[自然景物] + 穿过/飞过 + [场景]，[纯真画面]`
   → "鸟飞翔 穿过了这条小巷"

### C. 韵脚偏好
- 主韵：-ing（景/听/信/心/行/青/影）
- 辅韵：-an（年/天/边/间/脸/远）

### D. 节奏与结构特征
- 句式轻快明朗（6-9字为主）
- 叙事性强，像在讲一个故事
- 副歌以"约定"/"回忆"为关键词收束

### E. 参考金句
- "小学篱笆旁的蒲公英 是记忆里有味道的风景" ——《蒲公英的约定》
- "一起长大的约定 那样清晰 打过勾的我相信" ——《蒲公英的约定》
- "琥珀色黄昏像糖在很美的远方 你的脸没有化妆我却疯狂爱上" ——《园游会》
- "瓶中沙 写的话 问你是否还牵挂" ——《瓶中沙》
- "为什么上课时举手很难 为什么拿在线宝物简单" ——《三年二班》

## 风格7：hiphop_rap 嘻哈说唱

### A. 核心意象词库
- **态度类**：嚣张、拽、跩、地盘、style、flow、Bro、Homie、自信、个性
- **街头类**：篮球场、街舞、DJ、麦克风、beat、rap、街头、巷子、涂鸦
- **动作类**：说唱、喷、diss、battle、弹舌、freestyle、飙汗
- **混搭类**：中英文穿插、方言、俚语、网络用语、拟声词

### B. 经典句式模板
1. `[态度宣言]，[个人原则]`
   → "在我地盘这 你就得听我的"
2. `[短句连击] + [拟声/语气词] + [短句连击]`
   → "干什麽 干什麽 呼吸吐纳心自在"
3. `[英文词] + [中文态度]，[街头场景]`
   → "Say that you love me Say that you care"
4. `用 + [方式] + [动作]，[个人态度]`
   → "我用形意猴拳在练习 引你注意"

   → "我就是嚣张 我就是一代棋王"

### D. 节奏与结构特征

1. `如果 + [假设/挑战]，[文化/民族回应]`
   → "如果华佗再世 崇洋都被医治"
2. `我用 + [力量动作]，[改变/突破结果]`
   → "东亚病夫的招牌 已被我一脚踢开"
3. `[角色] + 不会/不能 + [弱点]，但我 + [正面宣言]`
   → "超人不会飞 但我可以保护你"
4. `让我来 + [创作/改变行为]，用 + [方式] + [表达]`
   → "让我来调配 用我的方式 改编成音乐"

### C. 韵脚偏好
- 主韵：-ang（强/刚/当/上/讲/扬/场）
- 辅韵：-ao（傲/药/效/道/高/豪）

### D. 节奏与结构特征
- 段落由蓄力到爆发（低沉主歌 → 高能副歌）
- 副歌口号感强，适合合唱
- 结尾常为宣言式收束

### E. 参考金句
- "如果华佗再世 崇洋都被医治 外邦来学汉字 激发我民族意识" ——《本草纲目》
- "超人不会飞 但我可以保护你" ——《超人不会飞》
- "放手一搏令谁都惭愧 迎着风极速在超越" ——《龙战骑士》

## 风格9：movie_ost 影视主题曲

### A. 核心意象词库
- **画面类**：镜头、画面、桥段、剧本、角色、剧情、远景、特写
- **时代类**：朝代、年代、民国、古代、战场、宫庭、乱世、盛世
- **情感类**：离别、等待、思念、宿命、使命、抉择、牺牲、忠义
- **场景类**：城墙、古道、长廊、大殿、江湖、河山、战场、城门

### B. 经典句式模板
1. `[画面描写] + [时代/场景]，[命运/情感]`
   → "菊花残满地伤 你的笑容已泛黄"
2. `[身份/角色] + [宿命/使命]，[个人情感]`
   → "我一身的戎装 呼啸沧桑"
3. `[时间/季节] + [环境变化]，[情感映射]`
   → "愁莫渡江 秋心拆两半"
4. `[宏大场景] + [个人渺小]，[等待/思念]`
   → "我送你离开 千里之外 你无声黑白"

### C. 韵脚偏好
- 随剧情变化换韵（平静段-ang，冲突段-an，收尾段-i）
- 副歌韵脚最稳定

### D. 节奏与结构特征
- 结构完整（Intro-Verse-Chorus-Bridge-Outro全部标注）
- 叙事起承转合清晰
- 常使用"闪回"/"跳接"式的时间叙事

### E. 参考金句
- "你的泪光 柔弱中带伤 惨白的月弯弯 勾住过往" ——《菊花台》
- "我送你离开 千里之外 你无声黑白" ——《千里之外》
- "屋檐如悬崖 风铃如沧海 我等燕归来" ——《千里之外》
- "愁莫渡江 秋心拆两半 怕你上不了岸 一辈子摇晃" ——《菊花台》

## 风格10：exotic_fusion 异域融合

### A. 核心意象词库
- **欧洲类**：布拉格、米兰、教堂、广场、哥德式、巴洛克、油画、琴键、彩绘玻璃
- **亚洲类**：樱花、忍者、居酒屋、神社、味噌汤、榻榻米、禅、武士
- **中东类**：苏美、巴比伦、法典、楔形文字、金字塔、沙漠、骆驼
- **美洲类**：牛仔、印地安、峡谷、草原、荒野、响尾蛇
- **艺术类**：吟唱诗人、民谣、手风琴、吉他、橱窗、酒馆

### B. 经典句式模板
1. `[异国地名] + 的 + [特色场景]，我 + [个人体验]`
   → "居酒屋里的小神龛 离铺满鹅卵石的玄关"
2. `[异国文化元素] + [动作/描写]，[个人情感/联想]`
   → "祭司 神殿 征战 弓箭 是谁的从前"
3. `[历史文化知识]，[浪漫/情感联系]`
   → "古巴比伦王颁布了汉摩拉比法典 刻在黑色的玄武岩"
4. `[异国建筑/艺术]，[画面/氛围]`
   → "巴洛克建筑的街道旁 一家烟雾缭绕的酒馆"

### C. 韵脚偏好
- 欧洲主题偏-an（场/窗/廊/堂/光）
- 亚洲主题偏-i（里/密/气/地/溪）
- 中东主题偏-en/-an（远/年/烟/言/前）

### D. 节奏与结构特征
- 长句铺陈（异域知识/背景）+ 短句抒情
- 常有念白/独白式段落
- 副歌回归情感核心

### E. 参考金句
- "古巴比伦王颁布了汉摩拉比法典 刻在黑色的玄武岩" ——《爱在西元前》
- "琴键上透着光 彩绘的玻璃窗 装饰着哥德式教堂" ——《布拉格广场》
- "居酒屋里的小神龛 离铺满鹅卵石的玄关" ——《忍者》
- "巴洛克建筑的街道旁 一家烟雾缭绕的酒馆" ——《米兰的小铁匠》
